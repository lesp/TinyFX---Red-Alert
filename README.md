# Where are the cool sound effects?

The cool sound effects are not here because they are copyrighted media.
If you want to source your own Star Trek audio files, then perhaps use a search engine and look for sound effects.

Remember to edit the files in an audio editor such as Audacity so that they are 16-bit mono WAV files. Also remember that TinyFX has only 4MB of flash storage.
